# Branding Prompts for Gemini

Use these exactly, then export PNG files in requested sizes.

## 1) icon.png (square)

Prompt:
"Create a clean app icon for a Web3 education mini app called Learn Base. Style: modern geometric, optimistic blue palette, white and navy accents, no text, no coin logos, no gradients that reduce legibility at small sizes. Visual idea: open book + chain link abstract symbol. Flat, high contrast, simple shapes, minimal details. Background transparent."

Export sizes:
- 1024x1024 PNG
- 512x512 PNG

## 2) cover.png (feed card)

Prompt:
"Design a mobile-first cover image for Learn Base mini app. Headline text: 'Learn Base Safely'. Subtext: '20 lessons from beginner to technical'. Include abstract educational motifs, chain network dots, and warm trustworthy color accents. Keep typography bold and readable on phone feed. Avoid clutter, avoid hype elements, avoid token price references."

Export size:
- 1200x630 PNG

## 3) splash.png (app launch)

Prompt:
"Create a splash screen image for Learn Base app launch. Minimal, centered icon symbol (book + chain), calm blue background, subtle light texture, no extra text except optional 'Learn Base'. Must look premium and load-friendly, simple composition."

Export size:
- 1024x1024 PNG

## 4) hero.png (manifest hero)

Prompt:
"Create a hero banner for Learn Base: educational, safe, beginner-friendly Web3 learning. Include subtle progress path visual with milestones (Basics, Safety, Technical). Colors: blue, white, soft amber. Keep clean whitespace and modern sans typography."

Export size:
- 1200x800 PNG

## 5) og.png (social preview)

Prompt:
"Create an Open Graph social image for Learn Base. Text: 'Learn Base' and 'Safe lessons. Real understanding.' Include icon and simple abstract network illustration. Professional, trustworthy, crisp contrast for social previews."

Export size:
- 1200x630 PNG

## Visual constraints

- No purple-heavy look.
- No meme styling.
- No fake charts/candlesticks.
- Avoid tiny text.
- Must stay readable on mobile.
